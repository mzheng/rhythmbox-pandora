==================
Pandora Plugin for Rhythmbox
==================

This is a Rhythmbox plugin based on Pithos_ that plays Pandora_ stations.

Features
=======
- Play / Pause / Next Song
- Creating/Deleting stations
- Thumbs Up / Thumbs Down / Tired of this song

Install
=======
	sh setup.sh
